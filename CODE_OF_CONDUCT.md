# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Wenn du Verhalten erlebst oder beobachtest, das gegen den Code of Conduct
verstößt, melde dich bei:

- **Email:** sr@rausch.se

Wir bestätigen den Eingang innerhalb von 72 Stunden, behandeln Reports
vertraulich und entscheiden nach Schwere des Vorfalls über eine
verhältnismäßige Reaktion (vom kurzen Hinweis bis zum dauerhaften Ausschluss
aus dem Projekt). Reporter werden über die Reaktion informiert.

Sicherheitsprobleme bitte stattdessen über
[`SECURITY.md`](SECURITY.md) melden.
